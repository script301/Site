Conversor de Mídia

v1.0.0

Descrição

O Conversor de Mídia é uma ferramenta simples e eficiente para converter arquivos de áudio e vídeo entre diversos formatos populares. Ideal para desenvolvedores, criadores de conteúdo e qualquer pessoa que precise adaptar arquivos de mídia para diferentes plataformas.

Funcionalidades

  • Conversão entre formatos de áudio (MP3, WAV, OGG, FLAC, etc.)
  • Conversão entre formatos de vídeo (MP4, AVI, MKV, MOV, etc.)
  • Extração de áudio de vídeos
  • Conversão em lote de múltiplos arquivos
  • Interface de linha de comando simples

Requisitos

  • Python 3.8+
  • FFmpeg instalado
  • Sistemas suportados: Windows, Linux, macOS

Instalação

# Clone o repositório
git clone https://github.com/seu-usuario/conversor-de-midia.git

# Instale as dependências
cd conversor-de-midia
pip install -r requirements.txt

# Execute o conversor
python converter.py --input arquivo.mp4 --output arquivo.mp3
